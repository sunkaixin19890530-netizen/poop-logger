# 💩 拉屎记录器 (Poop Logger)

一个简单的Web应用，帮助你记录和追踪自己的拉屎状态。

## 功能特点

- 📅 记录每次拉屎的日期和时间
- ⭐ 状态评分系统（1-5星）
- 📝 添加备注信息
- 💾 数据本地存储（localStorage）
- 📱 响应式设计，手机也能用

## 使用方法

直接在浏览器中打开 `index.html` 文件即可使用。

## 状态评分说明

- 🌟 5星 - 非常顺畅
- ✨ 4星 - 比较顺畅
- 😐 3星 - 一般
- 😣 2星 - 有点困难
- 💔 1星 - 非常困难

## 技术栈

- HTML5
- CSS3
- Vanilla JavaScript

## 数据存储

所有数据都存储在浏览器的localStorage中，不会上传到任何服务器。
